# Phase 1 快速开始指南

## 🚀 快速启动

### 1. 查看个人中心页面

启动开发服务器后，访问：
```
http://localhost:5173/hub
```

### 2. 导航方式

在左侧边栏中点击 **"Personal Hub"** 即可进入个人中心。

## 🔧 技术细节

### 使用的技术栈
- **React 19** + TypeScript
- **Zustand** - 状态管理
- **Tailwind CSS** - 样式
- **Lucide React** - 图标
- **React Router** - 路由

### 数据流
```
组件 → Store → Service → API (Mock)
                    ↓
                Mock 数据返回
```

### Mock 数据说明
所有 API 调用当前返回 Mock 数据，便于前端开发。
替换为真实 API 时，只需修改 `src/services/` 中的服务文件。
